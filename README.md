
# 👟 Bot Asistente de Zapatillas

Bot de IA que responde preguntas sobre zapatillas en tiempo real.

## Demo en vivo
🔗 https://huggingface.co/spaces/drkielztr/bot-zapatillas

## ¿Qué hace?
- Responde preguntas sobre modelos, precios y marcas
- Catálogo real de 114 productos de Footloose Perú
- Disponible 24/7 desde cualquier celular o PC

## Tecnologías
- Python + Groq API (LLaMA 3.3 70B)
- Gradio para la interfaz
- Playwright para scraping
- Pandas para análisis de datos
